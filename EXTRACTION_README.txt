MEMORY EXTRACTION INSTRUCTIONS
================================

1. Make sure your dev server is running: npm run dev
2. Open your browser to: http://localhost:5173
3. Open Developer Tools (F12 or right-click → Inspect)
4. Go to the Console tab
5. Copy and paste the entire contents of extract-script.js
6. Press Enter
7. Copy the output between "=== COPY THIS DATA ===" and "=== END COPY ==="
8. Send that data to the developer

That's it! No manual file handling required.
